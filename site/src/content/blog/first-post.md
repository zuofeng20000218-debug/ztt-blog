这篇文章是博客里的第一篇测试内容，用来记录网站正式上线前后的关键步骤。

目前这套博客使用了下面这组组合：

- Astro 作为静态站点框架
- GitHub 作为代码仓库
- Cloudflare Pages 负责自动部署
- `www.200302.xyz` 作为正式访问域名

从结果看，这套方案很适合个人博客：

- 成本低
- 访问速度还可以
- 更新方式简单
- 以后改主题和内容也比较灵活

这篇文章先保留为建站日志，后面我会继续补充域名配置、Pages 构建设置和 SSH 推送过程中的踩坑记录。
